🎣 Smart Fisheries

AI-Powered Marine Surveillance & Fisheries Management System

Smart Fisheries is an intelligent surveillance and fisheries management system developed to monitor coastal regions using Artificial Intelligence and Computer Vision. The system detects illegal fishing activities during government-declared trolling ban periods and identifies drowning incidents in real time, helping authorities improve marine safety and fisheries management.

📌 Overview

Traditional fisheries monitoring relies heavily on manual inspections and coastal patrols, making it difficult to monitor large coastal areas continuously.

Smart Fisheries automates this process by integrating:

Artificial Intelligence
Computer Vision
Django Web Framework
OpenCV
Flutter Mobile Application
MySQL Database

The system continuously analyzes video feeds from surveillance cameras and instantly alerts authorities when illegal fishing or drowning is detected.
