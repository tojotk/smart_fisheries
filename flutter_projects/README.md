# 📱 Smart Fisheries Mobile Application

The **Smart Fisheries Mobile Application** is the Flutter-based mobile client of the Smart Fisheries ecosystem. It enables fishermen, boat owners, and fisheries authorities to access services such as trip management, emergency reporting, notifications, weather updates, and safety information directly from their mobile devices.

This application works with the Smart Fisheries Django backend through REST APIs to provide real-time communication and fisheries management.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/tojotk/smart_fisheries.git
```

### Navigate to Flutter Project

```bash
cd smart_fisheries/flutter_projects
```

### Install Dependencies

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```
